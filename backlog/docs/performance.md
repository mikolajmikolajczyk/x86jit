## Tiering — wall-clock by mode

`eager` compiles every block on first execution; `tier` interprets a block until it is hot (50 runs) then compiles it inline; `bg` compiles hot blocks on a worker thread (`x86jit-run` ships `tier`). `best↓` is the fastest mode's speedup over `eager`.
